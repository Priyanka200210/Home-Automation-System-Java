A console-based **Home Automation System** developed in Java that allows users to manage rooms, add multiple devices, turn devices ON/OFF remotely, and track device usage time. The project is built using Object-Oriented Programming (OOP) principles such as classes, objects, inheritance, and polymorphism.

## Features
### Room Management
Supports multiple room types:  
Kitchen  
Living Area  
Dining Area  
Bedroom  

### Device Management
Add any device to any room
Supports future devices (extensible design)  
**Examples:**  
Lights  
Fans  
Air Conditioners  
Television  
Showers  
Electronic gadgets  

### Controls
Turn ON/OFF any device in any room  
Check device status  
Manage any number of devices  

### Usage Time Tracking
Displays how long a device has been ON or OFF  
Tracks total active time of each device  

### Technology Stack
Java  
Object-Oriented Programming (OOP)  
Console-based user interaction
